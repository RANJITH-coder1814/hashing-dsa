# hashing-dsa
A collection of hashing-based problems implemented in C++ to demonstrate efficient use of hash maps and hash sets for solving common DSA problems.
# 🔐 Hashing DSA (C++)

## 📖 Description
This repository contains common hashing problems solved using C++ STL (unordered_map).

## 🚀 Features
- Frequency Count of Elements
- Highest Occurring Element
- Two Sum Problem

## 🛠️ Technologies Used
- C++
- STL (unordered_map, vector)

## 📂 How to Run
1. Clone the repository
2. Open in any C++ IDE
3. Compile:
   g++ main.cpp -o hashing
4. Run:
   ./hashing
